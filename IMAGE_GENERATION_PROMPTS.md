## Usage Notes

- **Recommended Size**: 1200x630px (for GitHub social preview) or 1920x1080px (for README banner)
- **Format**: PNG with transparency or JPG
- **Style**: Modern, professional, tech-focused
- **Colors**: F1 red (#E10600), black, white, with blue/cyan for data elements

## Tools to Use

1. **Google Imagen** (if available)
2. **Midjourney** - Add `--ar 16:9` for banner ratio
3. **DALL-E 3** - Works well with detailed prompts
4. **Stable Diffusion** - Use with proper aspect ratio flags
5. **Adobe Firefly** - Good for professional branding
